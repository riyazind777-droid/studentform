# 🎓 Student Form - JavaScript CRUD Project

My first JavaScript project built while learning Frontend Development.

This project allows users to enter student details through a form and dynamically add them to a table using JavaScript DOM manipulation. It also includes the ability to delete individual student records.

## 🚀 Features

- Add student details dynamically
- Capture user input using JavaScript
- DOM Manipulation
- Event Handling with `addEventListener()`
- Delete individual rows
- Simple and responsive UI
- Beginner-friendly JavaScript project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 Concepts Practiced

- DOM Manipulation
- `addEventListener()`
- `createElement()`
- `appendChild()`
- `querySelector()`
- Form Input Handling
- Dynamic Table Creation
- Event Handling
- Functions
- Variables
